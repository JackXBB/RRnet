# 训练与可视化说明（论文展示版）

本项目已补充：

1. **预处理结果图自动保存**：默认保存到 `artifacts/preprocess/<subject_id>/`。
2. **训练曲线图自动保存**：每折训练后自动从 CSV 日志汇总绘制，默认保存到 `artifacts/training/fold_<fold_id>/训练曲线.png`。
3. **验证集/测试集分析图保存**：在 Lightning 日志目录下保存（含回归图、Bland-Altman 图、分段 MAE 图）。

---

## 一、预处理结果图

执行训练后，每个受试者会生成：

- `pipeline_timeseries.png`：原始信号、去噪后、带通滤波后、异常值处理后。
- `segment_scalogram.png`：示例片段与对应时频图。

默认路径：

```bash
artifacts/preprocess/<subject_id>/
```

---

## 二、训练结果图

每个 fold 训练完成后自动输出：

- `训练曲线.png`：
  - 左图：训练损失/验证损失随 epoch 变化
  - 右图：验证 MAE 随 epoch 变化

默认路径：

```bash
artifacts/training/fold_<fold_id>/训练曲线.png
```

---

## 三、各模型训练命令

> 以下命令在仓库根目录执行，默认入口是 `src/main.py`（Hydra 配置）。

### 1）融合模型（时域+频域）

```bash
python src/main.py training.ablation_mode=fusion training.model_name=Linear
```

### 2）仅时域模型（Time Only）

```bash
python src/main.py training.ablation_mode=time_only training.model_name=Linear
```

如需使用 LSTM：

```bash
python src/main.py training.ablation_mode=time_only training.model_name=LSTMRR
```

### 3）仅频域模型（Freq Only）

```bash
python src/main.py training.ablation_mode=freq_only
```

### 4）启用时域 SSL 预训练（仅 time_only 模式）

```bash
python src/main.py training.ablation_mode=time_only training.use_ssl_pretraining=true
```

### 5）启用频域 SSL 预训练（仅 freq_only 模式）

```bash
python src/main.py training.ablation_mode=freq_only training.use_ssl_pretraining=true
```

## 四、论文展示建议（中文图）

建议在论文中优先使用以下图片：

1. 预处理流程图：`pipeline_timeseries.png`
2. 时频表示图：`segment_scalogram.png`
3. 训练收敛图：`训练曲线.png`
4. 性能分析图：测试集回归图 + Bland-Altman 图 + 分段 MAE 图

除 `Bland-Altman`、`MAE`、`RR`、`Epoch` 等常见固定英文表达外，其余图标题和坐标轴文字已中文化。
